# Changelog

All notable changes to Home Page are documented in this file.

The project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-05

### Added

- Native-style Home dashboard opened from Discord's Direct Messages area.
- Native Direct Messages sidebar integration and selected Home state.
- Server search and sorting controls.
- Favorite Servers with drag-and-drop ordering.
- Recent Servers with automatic or fixed display counts and clearable history.
- Expandable Discord server folders.
- All Servers grid with icons, descriptions, online counts, and member totals.
- Separate unread and mention indicators and filters.
- In Voice and On Stage activity filters.
- Optional voice participant previews on server cards.
- Cached server descriptions with manual refresh support.
- Settings grouped into semantic sections with Discord-style switches.
- Support for Discord Light, Ash, Dark, and Onyx/Midnight themes.
- Compatibility handling for native window controls and third-party plugin popovers.
- Local persistence for settings, favorites, recent servers, and cached descriptions.

### Requirements

- BetterDiscord.
- BDFDB Library (`0BDFDB.plugin.js`).
